Welcome to ASD diagnosis assistant!

In this website, a series of pictures will be shown, and the eye-tracking data of the child will be collected via the web camera. The data collected will be used only for diagnostic and research purposes.

Note:

  1. Please place the web camera at the center, above the screen.
  2. Parents (or teachers) should stay away from the screen during calibration and testing.
  3. Chrome, Firefox and Edge are recommended. (Safari is not currently supported).

Website Instructions:

For each child:

  1. First we need to calibrate the camera. After the button is pressed, the page will redirect to the calibration page. Please help select the distance of the child from the screen and the size of the screen. Then ask the child to look at the screen during calibration.
  2. After calibration, please input the child's name and his/her ASD status. Fullscreen is recommended because it involves fewer distractions. Then press the continue button to start testing.
  3. During the test, several groups of pictures will be shown, and the child should look at the screen. After the test, the result will be uploaded to the server automatically. Then you can press the retest button to test the next child.
